
## Nomenclature du projet
- Keyboard  : Un clavier mécanique custom (clavier personnalisé)
- Inventory : L'inventaire personnel d'un membre
- Showcase : Une galerie publique de claviers

## Propriétés des entités

### Keyboard (Clavier)
- name (string) : Nom du clavier
- brand (string) : Marque (ex: Keychron, GMMK)
- switchType (string) : Type de switches (Cherry MX, Gateron...)
- keycapSet (string) : Set de keycaps
- description (text) : Description détaillée
- image (string, optionnel) : Photo du clavier

### Inventory
- description (string) : Description de l'inventaire

### Showcase
- name (string) : Nom de la showcase
- description (text) : Description
- published (boolean) : Publié ou non
